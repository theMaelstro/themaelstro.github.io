MHFZZ Database Released!

26 Aug 2022, theMaelstro

A Companion App port for Monster Hunter Frontier

Description.

FEATURES
  • Monster List (ailments, weakness, habitats, rewards and carves) Work in progress
  • Weapon List with details (reach, melodies, coatings etc.)
  • Armor Sets (stats summary: skills, def etc.) Work in progress
  • Armor Parts
  • Quest List (sorted in categories matching in game npcs, key quest flags, rewards and details)
  • Item List (search function, item combination including 3 slot recipes and alchemy skill)
  • Location List (monster habitats, item gathering info with night/day and season info)
  • Decoration List Work in progress
  • Skill List with active skills and its presence on deco/armor
  • Armor Set Buidler (use amulets like cuffs) Performance may vary!
  • Settings (display bloated values for weapon damage)
